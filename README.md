## ✨ על המרחב
דף נחיתה יוקרתי ומדויק עבור המותג **Ritual** של אביה שריד. האתר עוצב בקפידה כדי לשקף חווית טיפול הוליסטית, המשלבת טכניקה מדויקת עם אסתטיקה נקייה ואוורירית.

### Highlights:
- **דיוק. עדינות. נוכחות.** - שפה מותגית אחידה לכל אורך חווית הגלישה.
- **Multi-language Support** - תמיכה מלאה בעברית ובאנגלית כולל התאמת כיווניות (RTL/LTR).
- **Hollywood Carbon Peel** - סקציה ייעודית לטיפול הדגל עם פירוט יתרונות והנחיות בטיחות.
- **Smart Contact Logic** - מערכת יצירת קשר חכמה המנתבת פניות ישירות ל-WhatsApp של אביה.

---

## 🛠 ארגז כלים טכנולוגי
האתר נבנה בקוד נקי (Pure Code) כדי להבטיח ביצועים מקסימליים וטעינה מיידית:
- **Frontend:** HTML5, CSS3 (Modern Flexbox & Grid).
- **Interactions:** Vanilla JavaScript לניהול מצבי שפה ואנימציות גלילה.
- **UI Effects:** Glassmorphism (אפקט זכוכית חלבית) ו-Smooth Scroll.
- **Hosting:** הפצה דרך Vercel עם חיבור רציף ל-GitHub.
