## ✨ על הפרויקט
דף נחיתה יוקרתי ומדויק עבור מותג הביוטי **Ritual** של אביה שריד. האתר נבנה מתוך דגש על חווית משתמש אוורירית, אסתטיקה גבוהה ותמיכה מלאה בשפות (עברית ואנגלית).

### מה המרחב מציע:
- **עיצוב גבות והצללת פודרה** בטכניקות המתקדמות ביותר.
- **הסרת איפור קבוע בלייזר** וטיפולי Carbon Peel.
- **פילוסופיית יופי** המשלבת האטה, יישור ונצחיות.

---

## 🛠 טכנולוגיות
האתר נבנה בקוד נקי (Pure Code) ללא מערכות כבדות כדי להבטיח טעינה מהירה ודירוג SEO גבוה:
- **HTML5 & CSS3** - מבנה ועיצוב רספונסיבי.
- **JavaScript** - לוגיקת החלפת שפה (i18n) ואפקטים ויזואליים.
- **Glassmorphism UI** - שימוש באלמנטים שקופים למראה מודרני.
